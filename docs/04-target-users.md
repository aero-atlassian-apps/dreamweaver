# 04. Target Users

## Primary Personas

### 1. Sarah — The Exhausted Professional

| Attribute | Detail |
|-----------|--------|
| **Age** | 35 |
| **Occupation** | Marketing Manager |
| **Children** | Emma (6), Noah (4) |
| **Tech Comfort** | High |
| **Pain Points** | Too tired to read with enthusiasm, guilt about screen time |
| **Magic Moment** | Hearing Emma ask "Can we use DreamWeaver tonight?" |

**Journey:**
- 6:30 PM: Dinner chaos
- 7:30 PM: Bath time
- 8:00 PM: **BEDTIME RITUAL** — DreamWeaver active zone
- 8:30 PM: Memory review

**What She Needs:**
- One-tap story start
- AI suggestions (don't make her decide)
- Stories that lull kids to sleep
- Captured moments to review later

---

### 2. Michael — The Long-Distance Dad

| Attribute | Detail |
|-----------|--------|
| **Age** | 42 |
| **Occupation** | Consultant (travels 2 weeks/month) |
| **Children** | Sophia (5) |
| **Pain Points** | Misses bedtime, video calls feel awkward |
| **Magic Moment** | Voice-cloned story plays from hotel room |

**Journey:**
- Hotel room (8 PM local) → Records story request
- Story generates in Michael's voice
- Notification sent to Sophia's device at home
- Mom plays story for Sophia
- Michael reviews memory next morning

---

### 3. Linda — The Remote Grandparent

| Attribute | Detail |
|-----------|--------|
| **Age** | 68 |
| **Role** | Caregiver 2 days/week |
| **Tech Comfort** | Low |
| **Pain Points** | Doesn't know current kids' interests, feels out of touch |
| **Magic Moment** | Receives Grandma Mode link, hears grandchild's voice |

**What She Needs:**
- Simple one-tap access (no account required)
- Emotional hook (grandchild's actual voice)
- Beautiful, shareable moments

## User Segmentation

| Segment | % of Users | Key Feature |
|---------|------------|-------------|
| **Exhausted Professionals** | 45% | Voice clone, AI suggestions |
| **Long-Distance Parents** | 25% | Remote story, memory sharing |
| **Grandparents/Family** | 20% | Grandma Mode |
| **Engaged Parents** | 10% | Full feature usage |
