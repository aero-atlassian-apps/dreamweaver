# 27. COPPA Compliance

## Children's Online Privacy Protection Act

DreamWeaver handles data from children under 13. COPPA compliance is mandatory.

---

## Compliance Posture

| Requirement | Implementation | Status |
|-------------|----------------|--------|
| **Parental Consent** | Parent creates account, adds children | ✅ |
| **Privacy Notice** | Clear notice before collection | ✅ |
| **Limited Collection** | Only essential data (name, age, preferences) | ✅ |
| **Parental Access** | Export all child data on request | ✅ |
| **Parental Deletion** | Full deletion within 24 hours | ✅ |
| **Third-Party Limits** | No data sharing without consent | ✅ |
| **Security** | Encryption in transit and at rest | ✅ |

---

## Data Collected

| Data Type | Purpose | Retention |
|-----------|---------|-----------|
| Child's name | Personalization | Until deletion |
| Age | Age-appropriate content | Until deletion |
| Interests | Story recommendations | Until deletion |
| Voice (optional) | Golden moments | User-controlled |
| Story history | Memory vault | User-controlled |

---

## Parental Controls

| Control | Description |
|---------|-------------|
| **View Data** | See all data collected for each child |
| **Export Data** | Download complete data package |
| **Delete Data** | Immediate, full deletion |
| **Disable Features** | Turn off voice recording, sharing |

---

## Technical Safeguards

- **No direct child registration** — Parent account required
- **No behavioral advertising** — No ad targeting
- **No persistent identifiers** — No cross-site tracking
- **Encrypted storage** — All data encrypted at rest
- **Audit logging** — All data access logged
